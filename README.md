## Backend Básico para Proyectos con Usuarios y Roles

Este repositorio proporciona un backend fundamental para proyectos que requieren gestión de usuarios y roles. Está construido utilizando las siguientes tecnologías:

- **Express:** Un framework web para Node.js que facilita la creación de API robustas y escalables.

- **PostgreSQL:** Una base de datos relacional que ofrece rendimiento y confiabilidad para almacenar y recuperar datos.

- **CORS (Cross-Origin Resource Sharing):** Un middleware que permite o restringe las solicitudes de recursos desde un origen diferente al del servidor.

- **dotenv (.env):** Una herramienta que facilita la configuración de variables de entorno en el desarrollo de aplicaciones.

- **JWT (JSON Web Tokens):** Un estándar para la creación de tokens de acceso que se utilizan para autenticar y autorizar usuarios.

- **AJV (Another JSON Schema Validator):** Un validador de esquemas JSON que garantiza la integridad de los datos en la aplicación.

Este backend básico proporciona una base sólida para proyectos que requieren autenticación de usuarios, gestión de roles y una conexión segura con una base de datos PostgreSQL.
